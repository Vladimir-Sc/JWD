1. Opis vežbe
-------------------
Upoznavanje sa sa tehnologijom (Java Servlet) za dinamički generisane Web sadržaje, baziranoj na java programskom jeziku.  

2. Sadržaj vežbe
-------------------
Prinicp funckionisanja HTML protokola (HTML zahtev i HTML odgovor)
Statički i dinamički web sadržaj
Servlet klasa/e i njegove metode
Eclipse dinamički Web projekat (objašnjenje strukture) war file
WebServer - Tomcat struktura foldera 
Pokretanje pd Linux i Windows operativnim sistemom

5. Zadatak na času
-------------------
Zadatak 1 - HelloWorld servlet i njegov deploy na WebServer
Zadatak 2 - Realizovati prihvatanje parametara iz zahteva. Nakon toga pojedinacne parametre ispisati kao elemente html stranice
Zadatak 3 - Svaki unos (korisnicki HTTP zahtev) imena i prezimena (kao osobe) ispisati u html stranici.
Zadatak 4 - Napravti "poseban" Servlet koji prikazuje listu osoba na osnovu unetih imena i prezimena iz forme


6. Domaći zadatak
-------------------
Sve što ne stignete na času.
Zadatak 5 - Napraviti posebne servlte koji upisuju parne i neparne korisnike (po redosledu unosa)
Zadatak 6 - Napraviti servlet koji upisuje u fajl podatke o osobi (ime i prezime), napraviti servlet koji čita podatke iz fajla i ispisuje ih na ekran
Zadatak 7 - Naparaviti servlet koji simulira logovanje (unosi se korisničko ime i šifra), ako korisnik postoji prikazati stranicu za regstraciju vozila sa prvog časa, ako korisnik
ne postoji prikazati ponovo stranicu za logovanje ili stranicu koja ispisuje grešku prilikom logovanja (neispravno ime ili lozinka).
